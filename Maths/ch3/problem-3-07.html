Chapter 3 · Problem 7

Question

Use elementary row operations to find the inverse of A = [[1,2],[3,4]].

Background Concept — Elementary Operations

Elementary row operations are: (1) Interchange two rows, (2) Multiply a row by a non-zero constant, (3) Add a multiple of one row to another. To find A⁻¹, form the augmented matrix [A|I] and use elementary row operations to transform it to [I|A⁻¹].

Solution

Form [A|I] = [[1,2|1,0],[3,4|0,1]]. R₂ → R₂ - 3R₁: [[1,2|1,0],[0,-2|-3,1]]. R₂ → (-1/2)R₂: [[1,2|1,0],[0,1|3/2,-1/2]]. R₁ → R₁ - 2R₂: [[1,0|-2,1],[0,1|3/2,-1/2]]. Therefore A⁻¹ = [[-2,1],[3/2,-1/2]].

Tips to Solve

  • Form augmented matrix [A|I]
  • Use elementary row operations
  • Transform to [I|A⁻¹]
  • Check each step
  • Verify AA⁻¹ = I

Key Formulas

Use [A|I] → [I|A⁻¹] using elementary row operations
