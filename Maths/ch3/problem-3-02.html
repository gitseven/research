Chapter 3 · Problem 2

Question

If A = [[1,2],[3,4]] and B = [[5,6],[7,8]], find AB.

Background Concept — Matrix Multiplication

Matrix multiplication is defined as (AB)ᵢⱼ = Σₖ AᵢₖBₖⱼ. The number of columns in A must equal the number of rows in B. The result has the same number of rows as A and the same number of columns as B. Matrix multiplication is not commutative: AB ≠ BA in general.

Solution

AB = [[1×5+2×7, 1×6+2×8], [3×5+4×7, 3×6+4×8]] = [[5+14, 6+16], [15+28, 18+32]] = [[19, 22], [43, 50]].

Tips to Solve

  • Use (AB)ᵢⱼ = Σₖ AᵢₖBₖⱼ
  • Check dimensions: A is m×n, B is n×p
  • Result is m×p
  • Multiply row by column
  • Verify each element

Key Formulas

(AB)ᵢⱼ = Σₖ AᵢₖBₖⱼ; Dimensions: (m×n)(n×p) = (m×p)
