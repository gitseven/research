Question

Solve using Cramer's rule: 2x + 3y = 7, 4x + 5y = 13.

Background Concept — Cramer's Rule

Cramer's rule provides a method to solve systems of linear equations using determinants. For the system ax + by = e, cx + dy = f, the solutions are x = det([[e,b],[f,d]])/det([[a,b],[c,d]]) and y = det([[a,e],[c,f]])/det([[a,b],[c,d]]). This method works when the coefficient matrix is invertible.

Solution

The coefficient matrix is A = [[2,3],[4,5]] with det(A) = 2×5 - 3×4 = 10 - 12 = -2. For x: replace first column with constants: A₁ = [[7,3],[13,5]], det(A₁) = 7×5 - 3×13 = 35 - 39 = -4, so x = -4/(-2) = 2. For y: replace second column with constants: A₂ = [[2,7],[4,13]], det(A₂) = 2×13 - 7×4 = 26 - 28 = -2, so y = -2/(-2) = 1.

Tips to Solve

  • Find det(A) of coefficient matrix
  • Replace columns with constants
  • Calculate determinants of modified matrices
  • Use x = det(A₁)/det(A), y = det(A₂)/det(A)
  • Check the solution

Key Formulas

x = det(A₁)/det(A), y = det(A₂)/det(A) where A₁, A₂ are matrices with replaced columns
