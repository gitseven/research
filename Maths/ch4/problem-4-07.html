Question

Expand the determinant of A = [[2,1,0],[1,3,2],[0,1,1]] along the first row.

Background Concept — Determinant Expansion

The determinant can be expanded along any row or column using the formula det(A) = Σⱼ aᵢⱼCᵢⱼ where Cᵢⱼ is the cofactor. Expanding along the first row: det(A) = a₁₁C₁₁ + a₁₂C₁₂ + a₁₃C₁₃. This method is useful for larger matrices.

Solution

Expanding along first row: det(A) = 2C₁₁ + 1C₁₂ + 0C₁₃ = 2det([[3,2],[1,1]]) - 1det([[1,2],[0,1]]) + 0 = 2(3×1-2×1) - 1(1×1-2×0) = 2(3-2) - 1(1-0) = 2(1) - 1(1) = 2 - 1 = 1.

Tips to Solve

  • Use det(A) = Σⱼ aᵢⱼCᵢⱼ
  • Calculate each cofactor
  • Alternate signs: +, -, +
  • Check arithmetic
  • Verify with other methods

Key Formulas

det(A) = Σⱼ aᵢⱼCᵢⱼ; Cᵢⱼ = (-1)ᵢ⁺ʲMᵢⱼ
