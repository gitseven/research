Chapter 4 · Problem 8

Question

Find the volume of parallelepiped formed by vectors a = (1,2,3), b = (2,1,1), c = (3,2,1).

Background Concept — Volume of Parallelepiped

The volume of a parallelepiped formed by vectors a, b, c is |det([a;b;c])| where [a;b;c] is the matrix with a, b, c as rows. This formula uses the scalar triple product. The volume is zero if the vectors are coplanar (linearly dependent).

Solution

The volume is |det([[1,2,3],[2,1,1],[3,2,1]])| = |1(1×1-1×2) - 2(2×1-1×3) + 3(2×2-1×3)| = |1(1-2) - 2(2-3) + 3(4-3)| = |1(-1) - 2(-1) + 3(1)| = |-1 + 2 + 3| = |4| = 4.

Tips to Solve

  • Form matrix with vectors as rows
  • Find determinant
  • Take absolute value
  • Check if vectors are coplanar
  • Verify with scalar triple product

Key Formulas

Volume = |det([a;b;c])|; Scalar triple product = a·(b×c)
