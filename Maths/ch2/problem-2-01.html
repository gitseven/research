Chapter 2 · Problem 1

Question

Find the principal value of sin⁻¹(-1/2).

Background Concept — Principal Values

Inverse trigonometric functions are the inverse operations of trigonometric functions. The principal value is the value in the principal range: sin⁻¹: [-π/2, π/2], cos⁻¹: [0, π], tan⁻¹: (-π/2, π/2). These functions are one-to-one on their principal ranges.

Solution

We need to find θ such that sin θ = -1/2 and -π/2 ≤ θ ≤ π/2. Since sin(-π/6) = -1/2 and -π/6 ∈ [-π/2, π/2], the principal value is -π/6.

Tips to Solve

  • Use principal range for sin⁻¹: [-π/2, π/2]
  • Find angle with given sine value
  • Check if angle is in principal range
  • Use reference angles
  • Consider sign of the value

Key Formulas

sin⁻¹: [-π/2, π/2], cos⁻¹: [0, π], tan⁻¹: (-π/2, π/2)
