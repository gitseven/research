Chapter 1 · Problem 6

Question

For the binary operation * on Z defined by a * b = a + b - ab, find the identity element and inverse of element 3.

Background Concept — Identity and Inverse Elements

An identity element e for operation * satisfies a * e = e * a = a for all a. An inverse of element a is an element b such that a * b = b * a = e. Not all elements have inverses. The identity is unique if it exists. Inverses are unique if they exist.

Visualization

Solution

Identity: Let e be the identity. Then a * e = a, so a + e - ae = a, giving e - ae = 0, so e(1 - a) = 0. For this to hold for all a, we need e = 0. Check: a * 0 = a + 0 - a·0 = a ✓. Inverse of 3: Let 3 * x = 0, so 3 + x - 3x = 0, giving 3 + x(1 - 3) = 0, so 3 - 2x = 0, hence x = 3/2. But 3/2 ∉ Z, so 3 has no inverse in Z.

Tips to Solve

  • For identity: solve a * e = a
  • For inverse: solve a * x = e
  • Check if solution is in the set
  • Verify by substitution
  • Not all elements have inverses

Key Formulas

Identity: a * e = e * a = a; Inverse: a * a⁻¹ = a⁻¹ * a = e
