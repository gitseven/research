Chapter 1 · Problem 7

Question

Determine if f(x) = x³ - 3x is even, odd, or neither.

Background Concept — Even and Odd Functions

A function f is even if f(-x) = f(x) for all x in its domain. A function f is odd if f(-x) = -f(x) for all x in its domain. Even functions are symmetric about the y-axis. Odd functions are symmetric about the origin. Most functions are neither even nor odd. The sum of even functions is even, sum of odd functions is odd.

Visualization

Solution

f(-x) = (-x)³ - 3(-x) = -x³ + 3x = -(x³ - 3x) = -f(x). Since f(-x) = -f(x), the function is odd. This means the graph is symmetric about the origin.

Tips to Solve

  • Calculate f(-x)
  • Compare with f(x) and -f(x)
  • Even: f(-x) = f(x)
  • Odd: f(-x) = -f(x)
  • Neither: f(-x) ≠ f(x) and f(-x) ≠ -f(x)

Key Formulas

Even: f(-x) = f(x); Odd: f(-x) = -f(x)
