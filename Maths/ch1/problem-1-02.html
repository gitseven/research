Chapter 1 · Problem 2

Question

Let f: R → R be defined by f(x) = 2x + 3. Show that f is one-to-one and onto.

Background Concept — One-to-One and Onto Functions

A function f: A → B is one-to-one (injective) if f(x₁) = f(x₂) implies x₁ = x₂. It is onto (surjective) if for every y ∈ B, there exists x ∈ A such that f(x) = y. A function that is both one-to-one and onto is called bijective. Linear functions f(x) = ax + b with a ≠ 0 are bijective.

Visualization

Solution

One-to-one: Let f(x₁) = f(x₂). Then 2x₁ + 3 = 2x₂ + 3, so 2x₁ = 2x₂, hence x₁ = x₂. Therefore f is one-to-one. Onto: For any y ∈ R, we need x such that f(x) = y. So 2x + 3 = y, giving x = (y-3)/2. Since (y-3)/2 ∈ R for any y ∈ R, f is onto. Therefore f is bijective.

Tips to Solve

  • For one-to-one: assume f(x₁) = f(x₂) and show x₁ = x₂
  • For onto: for any y, find x such that f(x) = y
  • Linear functions f(x) = ax + b with a ≠ 0 are bijective
  • Check domain and codomain carefully
  • Bijective = one-to-one + onto

Key Formulas

One-to-one: f(x₁) = f(x₂) ⟹ x₁ = x₂; Onto: ∀y ∈ B, ∃x ∈ A such that f(x) = y
