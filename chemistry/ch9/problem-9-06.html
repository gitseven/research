Coordination Compounds · Problem 9.6

Original Question

Using IUPAC norms write the systematic names of the following: (i) [Co(NH₃)₆]Cl₃ (ii) [Pt(NH₃)₂Cl(NH₂CH₃)]Cl (iii) [Ti(H₂O)₆]³⁺ (iv) [Co(NH₃)₄Cl(NO₂)]Cl (v) [Mn(H₂O)₆]²⁺ (vi) [NiCl₄]²⁻ (vii) [Ni(NH₃)₆]Cl₂ (viii) [Co(en)₃]³⁺ (ix) [Ni(CO)₄]

Background Concept — IUPAC Naming Rules

IUPAC names for coordination compounds follow an order: (1) name ligands in alphabetical order (prefixes like di‑, tri‑ do not affect alphabetical position; for chelating ligands use bis‑, tris‑ when ligand name contains prefixes), (2) then the metal with oxidation state in Roman numerals in parentheses, (3) for anionic complexes, metal names end with “‑ate” using Latin names where available (ferrate for Fe, cuprate for Cu), (4) neutral ligands usually keep their names (ammine for NH₃, aqua for H₂O, carbonyl for CO), (5) nitrito‑N/nitrito‑O specify ambidentate binding, (6) counter‑ions are named separately after the complex cation (or before, for salts where complex is an anion). Apply oxidation number by balancing charges. Square planar vs tetrahedral geometry is not part of the name unless ambiguity exists, but is often inferred from known chemistry (e.g., [NiCl₄]²⁻ is usually tetrahedral).

Answer

(i) [Co(NH₃)₆]Cl₃ → hexaamminecobalt(III) chloride
(ii) [Pt(NH₃)₂Cl(NH₂CH₃)]Cl → diammine(chloro)(methylamine)platinum(II) chloride
(iii) [Ti(H₂O)₆]³⁺ → hexaquatitaniun(III) ion
(iv) [Co(NH₃)₄Cl(NO₂)]Cl → tetraammine(chloro)(nitrito‑O)cobalt(III) chloride
(v) [Mn(H₂O)₆]²⁺ → hexaaquamanganese(II) ion
(vi) [NiCl₄]²⁻ → tetrachloridonickelate(II)
(vii) [Ni(NH₃)₆]Cl₂ → hexaamminenickel(II) chloride
(viii) [Co(en)₃]³⁺ → tris(ethane‑1,2‑diamine)cobalt(III) ion
(ix) [Ni(CO)₄] → tetracarbonylnickel(0)

Tips to Solve

  • Alphabetize ligand names ignoring numerical prefixes.
  • Use “‑ate” for anionic complexes and Latin metal names where applicable.
  • For en, write tris(ethane‑1,2‑diamine) etc., not tris‑en.
