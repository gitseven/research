Coordination Compounds · Problem 9.24

Original Question

Write the IUPAC name for the following complexes and indicate the oxidation state, electronic configuration and coordination number. Also give stereochemistry and magnetic moment of the complexes: (i) K[Cr(H₂O)₂(C₂O₄)₂]·3H₂O (ii) [Co(NH₃)₅Cl]Cl₂ (iii) [CrCl₃(py)₃] (iv) Cs[FeCl₄] (v) K₄[Mn(CN)₆]

Background Concept — Systematic Parsing Workflow

For each complex: (1) determine the complex ion charge from counter‑ions; (2) deduce oxidation state (OS) of the metal using ligand charges (H₂O 0; py 0; NH₃ 0; oxalate C₂O₄²⁻ −2; Cl⁻ −1; CN⁻ −1); (3) compute d‑electron count from atomic configuration minus OS; (4) count coordination number (CN) by donor sites (bidentate oxalate counts 2; others listed are monodentate); (5) infer common stereochemistry (octahedral CN 6; tetrahedral CN 4; square planar rare for given sets); (6) estimate spin state via ligand field (CN⁻ strong, oxalate moderate, NH₃ moderate, H₂O weak, Cl⁻ weak) and geometry; (7) compute spin‑only μ ≈ √[n(n+2)] BM from number of unpaired electrons n. This method provides consistent, defensible assignments suitable for problem solving.

Answer

(i) K[Cr(H₂O)₂(C₂O₄)₂]·3H₂O

  • Name: potassium diaquabis(oxalato)chromate(III) trihydrate
  • OS: Cr +3 (complex is −1; ligands: 2×ox −4, 2×H₂O 0 ⇒ Cr +3)
  • d count: d³; CN: 2 (H₂O) + 2×2 (ox) = 6; stereo: octahedral
  • Spin/μ: d³ (t₂g³) → 3 unpaired; μ ≈ √15 ≈ 3.87 BM

(ii) [Co(NH₃)₅Cl]Cl₂

  • Name: pentaamminechloridocobalt(III) chloride
  • OS: Co +3 (complex 2+ with 2 Cl⁻ outside)
  • d count: d⁶; CN: 6; stereo: octahedral
  • Spin/μ: Often low‑spin with NH₃ (depending on Co³⁺) → 0 unpaired; μ ≈ 0 BM

(iii) [CrCl₃(py)₃]

  • Name: trichloridotripyridinechromium(III)
  • OS: Cr +3; d count: d³; CN: 6 (3 Cl⁻, 3 py); stereo: octahedral
  • Spin/μ: d³ → 3 unpaired; μ ≈ 3.87 BM

(iv) Cs[FeCl₄]

  • Name: caesium tetrachloridoferrate(III) or (II) depending on charge; complex is −1 ⇒ Fe +3 (since 4Cl⁻ = −4; x−4 = −1 ⇒ x = +3)
  • d count: Fe³⁺ d⁵; CN: 4; stereo: tetrahedral likely
  • Spin/μ: Tetrahedral, weak‑field Cl⁻ → high‑spin d⁵ (5 unpaired); μ ≈ 5.92 BM

(v) K₄[Mn(CN)₆]

  • Name: potassium hexacyanidomanganate(II)
  • OS: Mn +2 (complex −4; 6CN⁻ = −6 ⇒ Mn +2); d count: d⁵
  • CN: 6; stereo: octahedral; Spin/μ: CN⁻ strong‑field often low‑spin d⁵ (1 unpaired) → μ ≈ √3 ≈ 1.73 BM

Tips to Solve

  • Find complex charge from counter‑ions, then OS.
  • Use ligand charges and denticity to get CN and geometry.
  • Estimate spin from spectrochemical series; compute μ from n.
