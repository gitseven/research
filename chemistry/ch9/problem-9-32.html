Coordination Compounds · Problem 9.32

Original Question

The correct order of wavelengths of absorption in the visible region for the complexes [Ni(NO₂)₆]⁴⁻, [Ni(NH₃)₆]²⁺ and [Ni(H₂O)₆]²⁺ is: (i) [Ni(NO₂)₆]⁴⁻ < [Ni(NH₃)₆]²⁺ < [Ni(H₂O)₆]²⁺ (ii) [Ni(NO₂)₆]⁴⁻ < [Ni(H₂O)₆]²⁺ < [Ni(NH₃)₆]²⁺ (iii) [Ni(H₂O)₆]²⁺ < [Ni(NH₃)₆]²⁺ < [Ni(NO₂)₆]⁴⁻ (iv) [Ni(NH₃)₆]²⁺ < [Ni(H₂O)₆]²⁺ < [Ni(NO₂)₆]⁴⁻

Background Concept — Same Reasoning as 9.31

Using the spectrochemical series (H₂O < NH₃ < NO₂⁻) and the relation λ ∝ 1/Δ₀, the order of absorption wavelengths from shortest to longest is NO₂⁻ < NH₃ < H₂O. Therefore the correct increasing order of λ is [Ni(NO₂)₆]⁴⁻ < [Ni(NH₃)₆]²⁺ < [Ni(H₂O)₆]²⁺.

Answer

Correct option: (i)
