d- and f-Block Elements · Problem 8.16

Original Question

Describe the oxidising action of potassium permanganate and write the ionic equations for its reaction with: (i) iodide (ii) iron(II) solution and (iii) H₂S.

Background Concept — Permanganate as Versatile Oxidant

Potassium permanganate (KMnO₄) is a powerful oxidising agent that can function in acidic, neutral, and basic media, with different reduction products depending on pH. In acidic solution, MnO₄⁻ reduces to Mn²⁺ (E° = +1.51 V), while in neutral/weakly basic solution it forms MnO₂, and in strongly basic solution it produces MnO₄²⁻. The acidic reduction half-reaction is: MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O. This makes permanganate more versatile than dichromate, which only works effectively in acidic medium. The deep purple color of MnO₄⁻ provides a clear visual indicator during titrations, disappearing when the reaction is complete. Permanganate can oxidise a wide range of species including halides, transition metal ions, sulfides, and organic compounds. The stoichiometry depends on the number of electrons transferred: 5 electrons per MnO₄⁻ ion in acidic medium. This versatility makes KMnO₄ a standard reagent in redox titrations and analytical chemistry.

Answer

Ionic equations in acidic medium:

(i) 2MnO₄⁻ + 16H⁺ + 10I⁻ → 2Mn²⁺ + 8H₂O + 5I₂
(ii) MnO₄⁻ + 8H⁺ + 5Fe²⁺ → Mn²⁺ + 4H₂O + 5Fe³⁺
(iii) 2MnO₄⁻ + 16H⁺ + 5H₂S → 2Mn²⁺ + 8H₂O + 5S

Tips to Solve

  • Use the acidic reduction half: MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O.
  • Oxidation halves: 2I⁻ → I₂ + 2e⁻; Fe²⁺ → Fe³⁺ + e⁻; H₂S → S + 2H⁺ + 2e⁻.
  • Scale to balance electrons: multiply oxidation half by 5, reduction half by 2 for iodide and sulfide.
  • For Fe²⁺, multiply oxidation half by 5 and use reduction half as-is.
