d- and f-Block Elements · Problem 8.36

Original Question

Write down the number of 3d electrons in each of the following ions: Ti²⁺, V²⁺, Cr³⁺, Mn²⁺, Fe²⁺, Co²⁺, Ni²⁺ and Cu²⁺. Indicate how would you expect the 3d orbitals to be occupied for these hydrated ions (octahedral).

Background Concept — 4s Lost Before 3d; High-spin Aqua Ions

For first-row transition cations, 4s electrons are removed before 3d. Hydrated (aqua) ions in octahedral fields with H₂O (a weak ligand) generally form high-spin complexes. Octahedral splitting yields lower-energy t₂g (3 orbitals) and higher-energy e_g (2 orbitals). In high-spin situations, electrons singly occupy all five d-orbitals before pairing, consistent with Hund’s rule and small Δ₀ relative to pairing energy for weak fields.

Answer

  • Ti²⁺: d² → t₂g² e_g⁰ (both unpaired)
  • V²⁺: d³ → t₂g³ e_g⁰ (all unpaired)
  • Cr³⁺: d³ → t₂g³ e_g⁰ (all unpaired)
  • Mn²⁺: d⁵ → t₂g³ e_g² (all unpaired; high-spin)
  • Fe²⁺: d⁶ → t₂g⁴ e_g² (high-spin; 4 unpaired)
  • Co²⁺: d⁷ → t₂g⁵ e_g² (high-spin; 3 unpaired)
  • Ni²⁺: d⁸ → t₂g⁶ e_g² (high-spin; 2 unpaired)
  • Cu²⁺: d⁹ → t₂g⁶ e_g³ (Jahn–Teller distortion likely; 1 unpaired)
Ordering rule (weak field, octahedral): fill t₂g singly → fill e_g singly → pair as needed.
