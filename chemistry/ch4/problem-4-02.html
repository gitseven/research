Chapter 4 · Problem 2

Question

The rate of reaction A + B → C is given by Rate = k[A]²[B]. What is the order of reaction with respect to A, B, and overall order?

Background Concept — Order of Reaction

The order of reaction with respect to a reactant is the power to which its concentration is raised in the rate law. The overall order is the sum of all individual orders. For Rate = k[A]ᵐ[B]ⁿ, the order with respect to A is m, with respect to B is n, and the overall order is m + n. The order must be determined experimentally.

Visualization

Solution

Given Rate = k[A]²[B]. Order with respect to A = 2 (exponent of [A]). Order with respect to B = 1 (exponent of [B]). Overall order = 2 + 1 = 3. Therefore, the reaction is second order with respect to A, first order with respect to B, and third order overall.

Tips to Solve

  • Identify the exponents in the rate law
  • Order with respect to each reactant = its exponent
  • Overall order = sum of all exponents
  • Check the units of rate constant
  • Verify with experimental data

Key Formulas

For Rate = k[A]ᵐ[B]ⁿ: Order w.r.t. A = m, Order w.r.t. B = n, Overall order = m + n
