Chemical Kinetics · Problem 4.18

Original Question

For a first order reaction, show that time required for 99% completion is twice the time required for the completion of 90% of reaction.

Background Concept — Fractional Completion in First Order

For first‑order kinetics, the integrated law is ln([A]₀/[A]) = kt. If a fraction x of the reaction has completed, the remaining fraction is (1−x), so ln(1/(1−x)) = kt. Thus, the time to reach any specified completion x is t(x) = (1/k) ln(1/(1−x)). This functional form depends only on x and k, not on [A]₀. Comparing two completions x₁ and x₂ gives a simple ratio t(x₂)/t(x₁) = ln[1/(1−x₂)] / ln[1/(1−x₁)]. For 90% completion, x = 0.90 ⇒ factor ln(10); for 99% completion, x = 0.99 ⇒ factor ln(100). Because ln(100) = 2 ln(10), the time to reach 99% is exactly twice the time to reach 90%, independent of k. This result generalises: moving one additional “nine” of completion multiplies time by the same constant factor because each step corresponds to a fixed increase in ln(1/(1−x)).

Answer

t₉₀ = (1/k) ln(10); t₉₉ = (1/k) ln(100) = (1/k)·2 ln(10) = 2 t₉₀

Tips to Solve

  • Use t(x) = (1/k) ln(1/(1−x)) for first‑order completion times.
  • Recall ln(100) = 2 ln(10) to show the 2× relationship.
  • Independence from [A]₀ is a hallmark of first‑order kinetics.
