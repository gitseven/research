Chemical Kinetics · Problem 4.26

Original Question

The decomposition of hydrocarbon follows the equation k = (4.5 × 10¹¹ s⁻¹) e^{−28000K/T}. Calculate Eₐ.

Background Concept — Comparing Exponent Forms

The Arrhenius form k = A e^{−Eₐ/(RT)} is sometimes written with a temperature‑scaled exponent constant: k = A e^{−C/T}. Equating exponents shows C = Eₐ/R. Therefore, if C is given (here 28000 K), Eₐ is obtained by multiplying by R in matching units. Using R = 8.314 J mol⁻¹ K⁻¹ yields Eₐ in J mol⁻¹; divide by 1000 for kJ mol⁻¹. This conversion bypasses any need for a specific temperature since C encodes the activation barrier per unit R.

Answer

Eₐ = C · R = 28000 K × 8.314 J mol⁻¹ K⁻¹ = 2.33 × 10⁵ J mol⁻¹ = 233 kJ mol⁻¹

Tips to Solve

  • Match exponents: −Eₐ/(RT) ≡ −C/T → Eₐ = CR.
  • Report Eₐ in kJ mol⁻¹ for readability.
  • Ensure the R value’s units align with desired Eₐ units.
