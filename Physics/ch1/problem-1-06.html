Chapter 1 · Problem 6

Question

Three students measure the same object and get: Student A: 2.1 cm, 2.0 cm, 2.2 cm; Student B: 2.5 cm, 2.6 cm, 2.4 cm; Student C: 2.3 cm, 2.3 cm, 2.3 cm. Which student is most precise? Most accurate? (True value = 2.3 cm)

Background Concept — Precision and Accuracy

Precision refers to consistency of measurements (how close measurements are to each other), while accuracy refers to how close measurements are to the true value. High precision means low scatter, high accuracy means close to true value. Both are important in experimental physics.

Visualization

Solution

Student A: average = 2.1 cm (accurate), range = 0.2 cm (moderate precision). Student B: average = 2.5 cm (inaccurate), range = 0.2 cm (moderate precision). Student C: average = 2.3 cm (accurate), range = 0 cm (most precise). Student C is most precise and accurate.

Tips to Solve

  • Calculate average for accuracy
  • Calculate range for precision
  • Compare with true value
  • Smaller range = higher precision
  • Closer to true value = higher accuracy

Key Formulas

Average = Σx/n, Range = max - min
